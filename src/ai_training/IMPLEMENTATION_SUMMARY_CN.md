## 🎯 核心改进：纠缠状态系统

**纠缠状态** = 粘合度（静态距离）+ 发散速度（动态变化）

这是本次改进的核心创新，来自FEATURE_ANALYSIS.md问题0。

### 三组纠缠状态特征

#### 1️⃣ 价格均线纠缠状态（2个特征）
```
特征码：F01_11, F01_12
名称：MA5-MA25粘合度、MA5-MA25发散速度
含义：短期均线与中期均线的粘合程度和发散速度
应用：捕捉金叉/死叉前兆，判断反转时机
```

#### 2️⃣ 成交量均线纠缠状态（2个特征）
```
特征码：F03_09, F03_10
名称：量线MA5-MA60粘合度、量线MA5-MA60发散速度
含义：成交量短期均线与长期均线的粘合程度和发散速度
应用：判断资金活跃度变化，识别成交量放缩趋势
```

#### 3️⃣ MACD动量纠缠状态（1个特征）
```
特征码：F02_09
名称：DIF-DEA发散速度
含义：MACD两条线的分离速度
应用：判断动能强度，预判MACD反转
```

## 📝 特征编码规则

### 格式说明
```
F###_##：特征群组编号-序号
├─ 第一个数字(3位)：群组编号
│  ├─ F01: 价格均线特征（14个）
│  ├─ F02: MACD特征（10个）
│  ├─ F03: 成交量特征（12个）
│  ├─ F04: 波动率特征（4个）
│  ├─ F05: 相对强度特征（5个）
│  ├─ F06: 强势特征（4个）
│  ├─ F07: 量能特征（3个）
│  ├─ F08: 趋势特征（2个）
│  └─ F09: 支撑阻力特征（2个）
│
└─ 后两个数字(2位)：组内序号（01-14）

优先级标签：[P0/P1/P2] 表示优先级（P0最高）
类型标签：[ENTANGLEMENT/CANDLE/MACD-ADV等] 表示特征类型
```

### 示例
- `F01_11: MA5-MA25粘合度 [P0-ENTANGLEMENT-001]`
  - F01_11：第1组第11个特征
  - P0：最高优先级
  - ENTANGLEMENT-001：纠缠状态系统的第1个特征
