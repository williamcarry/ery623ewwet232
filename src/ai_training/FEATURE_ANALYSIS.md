## ⚠️ 问题分析

### 问题0：缺少核心“纠缠状态”特征 ⭐⭐⭐⭐⭐ 最高优先级（新增）

**现状：**
现有特征只有静态的距离/比值，缺少动态的“纠缠状态”判断：

```python
# 现有：静态距离
'MA5>MA25'                 # 仅 0/1，无法知道差距多大
'DIF-DEA距离归一化'        # 静态距离，不知道趋势
'量线MA5>MA60'             # 仅 0/1，不知道放缩量程度

# 缺失：动态纠缠状态
- MA5-MA25的粘合度与发散速度
- 量线MA5-MA60的粘合度与发散速度
- DIF-DEA的发散速度
```

**影响：**
- 无法提前捕捉金叉/死叉前兆（极度粘合 + 快速粘合 = 交叉即将发生）
- 无法判断趋势强度（快速发散 = 趋势强，快速粘合 = 趋势弱）
- 无法识别变盘时机（价格 + 成交量 + MACD 三重粘合 = 大变盘在即）
- 错失最关键的动量信号

**改进方案：**

#### 第1组：MA5-MA25纠缠状态（+2个）- 已实现

**特征码：**
- `F01_11`: MA5-MA25粘合度 [P0-ENTANGLEMENT-001]
- `F01_12`: MA5-MA25发散速度 [P0-ENTANGLEMENT-002]

```python
# 新增特征：'F01_11: MA5-MA25粘合度'
粘合度 = |MA5 - MA25| / MA25
# 含义：
#   < 0.005  → 极度粘合（金叉/死叉即将发生）
#   0.005-0.01 → 粘合（可能反转）
#   0.01-0.02 → 正常分离
#   > 0.02    → 强势发散

# 新增特征：'MA5-MA25发散速度'
发散速度 = (粘合度[i] - 粘合度[i-5]) / (粘合度[i-5] + 0.0001)
# 含义：
#   > 0.5  → 快速分散（趋势强化加速）
#   < -0.5 → 快速粘合（反转信号）

# 代码位置：extract_features_sequence_from_kline_data() 第370-396行
```

#### 第2组：量线MA5-MA60纠缠状态（+2个）- 已实现

**特征码：**
- `F03_09`: 量线MA5-MA60粘合度 [P0-ENTANGLEMENT-004]
- `F03_10`: 量线MA5-MA60发散速度 [P0-ENTANGLEMENT-005]

```python
# 新增特征：'F03_09: 量线MA5-MA60粘合度'
量线粘合度 = |MA5_vol - MA60_vol| / MA60_vol
# 含义：
#   < 0.2  → 成交量平稳（横盘）
#   > 0.5  → 明显放量/缩量
#   > 1.0  → 异常放量/缩量

# 新增特征：'量线MA5-MA60发散速度'
量线发散速度 = (量线粘合度[i] - 量线粘合度[i-5]) / (量线粘合度[i-5] + 0.0001)
# 含义：
#   > 0 → 放量加速 或 缩量加速
#   < 0 → 成交量趋于平稳

# 代码位置：extract_features_sequence_from_kline_data() 第426-487行
```

#### 第3组：DIF-DEA纠缠状态（+1个）- 已实现

**特征码：**
- `F02_04`: DIF-DEA距离归一化 [粘合度，已存在]
- `F02_09`: DIF-DEA发散速度 [P0-ENTANGLEMENT-003]

```python
# 现有：'F02_04: DIF-DEA距离归一化' 即为 粘合度，可复用

# 新增特征：'F02_09: DIF-DEA发散速度'
DIF-DEA发散速度 = (粘合度[i] - 粘合度[i-5]) / (粘合度[i-5] + 0.0001)
# 含义：
#   > 0   → 动能加速
#   < -0.5 → 动能快速衰减（反转信号）

# 代码位置：extract_features_sequence_from_kline_data() 第398-424行
```

#### 第4组：三重纠缠联合判断（+2个，可选）
```python
# 新增特征：'三重粘合信号'
if (MA5-MA25粘合度 < 0.01 and 
    量线MA5-MA60粘合度 < 0.2 and 
    DIF-DEA粘合度 < 0.001):
    三重粘合信号 = 1  # 大变盘在即！

# 新增特征：'三重发散信号'
if (MA5-MA25发散速度 > 0.3 and 
    量线MA5-MA60发散速度 > 0.3 and 
    DIF-DEA发散速度 > 0.3):
    三重发散信号 = 1  # 趋势极强！

# 代码位置：extract_features_sequence_from_kline_data() 第650行后
```

**预期收益：** +3-4% 准确率（最高优先级）

**核心价值：**
- 粘合度：静态距离，判断当前状态
- 发散速度：动态变化，判断未来趋势
- 三重纠缠：价格 + 成交量 + MACD 共振，形成更强信号

#### 问题 7.2：未统一特征范围

**问题：** 特征范围不一致
```python
'布林带位置' → [0, 1]
'价格相对MA25强度' → [-∞, +∞]  # 无界
'RSI归一化' → [0, 1]
'价格加速度' → 无界
```

**改进方案：**
```python
# 在特征提取完成后，统一 Normalize 到 [-1, 1]
def normalize_features(features):
    """将所有特征归一化到 [-1, 1] 范围"""
    # 使用 tanh 或 sigmoid 函数确保有界性
    return np.tanh(features / np.std(features, axis=0))
```

## 📌 注意事项

1. **数据依赖：** 新增特征可能需要更多历史K线数据（建议保持 120+ 根）
2. **计算复杂度：** 增加特征数量会增加计算时间，建议监控性能
3. **过拟合风险：** 特征数量过多可能导致过拟合，建议配合特征选择算法
4. **向后兼容：** 修改特征时，需要重新训练所有已有的模型
