# 新增特征：'F03_09: 量线MA5-MA60粘合度'

#   < 0.2  → 成交量平稳（横盘，变盘在即）

#   > 0 → 放量加速 或 缩量加速（资金活跃度变化加剧）
#   < 0 → 成交量趋于平稳（横盘整理）

# 纠缠状态判断示例：
if 量线粘合度 < 0.2 and MA5-MA25粘合度 < 0.01:
    # 价格和成交量都横盘 → 变盘在即！

    # 持续放量 + 价格上涨 → 健康上涨！
